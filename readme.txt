1. File upload-keystore.jks
File ini adalah "Jantung"-nya aplikasi kamu.

Wajib: Simpan file ini di tempat aman (Laptop + Backup di Google Drive).

Risiko: Jika file ini hilang, kamu tidak akan bisa mengupdate aplikasi di Play Store selamanya (harus buat aplikasi baru dengan nama paket beda).

2. Password Keystore
Password yang tadi kamu ketik (dan tekan Enter).

Wajib: Ingat baik-baik (tadi kalau tidak salah dylearn123 atau sesuai yang kamu ketik).

Risiko: Punya file tapi lupa password = File tidak bisa dibuka = Tidak bisa update aplikasi.

3. Key Alias
Nama alias kunci yang kita pakai tadi.

Alias: upload
